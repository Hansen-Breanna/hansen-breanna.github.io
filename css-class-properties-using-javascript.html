Adding a Class

There are two ways to add a new class to an element. The first is classList.add("newClass") and the second is classList += "newClass".

Class Not Added
var createFirstClass = 
    document.getElementById("addClassOne");
createFirstClass.classList.add("classOne");
createFirstClass.textContent = 
    ".classOne added using .add()";
Class Not Added
var createSecondClass = 
    document.getElementById("addClassTwo");
createSecondClass.classList += "classTwo";
createSecondClass.textContent = 
    ".classTwo added using +=";

Removing Class Names

There are two ways to remove a class from an element. The first is classList.remove("newClass") and the second is classList -= "newClass".
